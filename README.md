# mental-health-screen-time
Project analyzing impact of daily screen time on mental health using Kaggle dataset.

# Mental Health and Screen Time Analysis

This project analyzes the relationship between daily screen time and self-reported mental health
using the **Mental Health and Technology Usage Dataset** from Kaggle.

## Repository Structure

- `data/` – instructions for obtaining the Kaggle dataset
- `notebooks/` – Jupyter notebooks for Milestone 1–3
- `reports/` – written reports for Milestone 2 and Milestone 3
- `src/` – (optional) Python scripts for reusable code
- `requirements.txt` – list of Python dependencies
